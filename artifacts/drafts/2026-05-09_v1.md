# OpenAI 推出 GPT-5.5 网络可信访问，测试 ChatGPT 广告，并发布实时语音模型

> 日期：2026-05-09

> OpenAI 发布 GPT-5.5 及 GPT-5.5-Cyber 并扩展网络安全可信访问，同时在 ChatGPT 中开始测试广告，推出新实时语音 API 模型；社区贡献 CyberSecQwen-4B 与 EMO 预训练混合专家模型。

## 要闻

- **#1 OpenAI 扩展网络安全可信访问，推出 GPT-5.5 和 GPT-5.5-Cyber** — OpenAI 扩大网络安全可信访问，推出 GPT-5.5 与 GPT-5.5-Cyber，帮助验证防御者加速漏洞研究与关键基础设施保护。  
  [https://openai.com/index/gpt-5-5-with-trusted-access-for-cyber](https://openai.com/index/gpt-5-5-with-trusted-access-for-cyber) _(source: openai_news)_
- **#2 OpenAI 发布新实时语音模型 API** — OpenAI API 新增实时语音模型，支持推理、翻译与转录，带来更自然、更智能的语音交互体验。  
  [https://openai.com/index/advancing-voice-intelligence-with-new-models-in-the-api](https://openai.com/index/advancing-voice-intelligence-with-new-models-in-the-api) _(source: openai_news)_

## 模型发布

- **#3 CyberSecQwen-4B：为防御性网络量身的小型专业模型** — 发布 CyberSecQwen-4B，专为防御性网络设计的小型、专业化、本地可运行模型，突显小模型在网络安全中的价值。  
  [https://huggingface.co/blog/lablab-ai-amd-developer-hackathon/cybersecqwen-4b](https://huggingface.co/blog/lablab-ai-amd-developer-hackathon/cybersecqwen-4b) _(source: huggingface_blog)_
- **#4 EMO：预训练混合专家模型实现涌现模块化** — 提出 EMO 预训练混合专家方法，通过预训练促成模型的涌现模块化，提升效率与表现。  
  [https://huggingface.co/blog/allenai/emo](https://huggingface.co/blog/allenai/emo) _(source: huggingface_blog)_
